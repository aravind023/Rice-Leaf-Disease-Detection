
# 🌾 Rice Leaf Disease Detection

## 📌 Project Overview

This project uses Deep Learning (MobileNetV2 Transfer Learning)
to classify rice leaf diseases.

## Diseases

- Bacterial Leaf Blight
- Brown Spot
- Leaf Smut

## Model

- MobileNetV2
- TensorFlow/Keras
- Transfer Learning

## Accuracy

Validation Accuracy: ~87%

## Features

- Upload rice leaf image
- AI disease prediction
- Confidence score
- Disease information
